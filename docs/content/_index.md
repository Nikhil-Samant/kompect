Kompect is a simple electron application build along with Vue and Vuetify. It uses Imagemin as its engine to compress images.

### Supported Images
- [x] JPG
- [x] PNG
- [ ] SVG
- [ ] WEBP
- [ ] GIF

> Note: The application is not tested throughly. Please use it at your own risk
